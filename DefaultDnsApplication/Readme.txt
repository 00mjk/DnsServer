Default DNS App for Technitium DNS Server
=========================================

This app requires MaxMind GeoIP2 database and includes the GeoLite2 version of the City and Country database for trial.

For production usage, it is required that you purchase the GeoIP2 database from MaxMind (https://www.maxmind.com/) then download and copy the GeoIP2-City.mmdb and GeoIP2-Country.mmdb files in to this app zip file and install the new zip file as your DNS app.
